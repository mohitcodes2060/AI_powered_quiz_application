# AI Quiz App

A complete React application that presents a series of multiple-choice questions and provides personalized AI feedback using the Gemini API.

## Features

- **Interactive Quiz**: Three multiple-choice questions with automatic progression
- **Progress Tracking**: Visual progress bar and question counter
- **AI Feedback**: Personalized feedback generated by Gemini AI based on user performance
- **Modern UI**: Clean, responsive design with Tailwind CSS
- **Score Display**: Final score with encouraging messages
- **Answer Review**: Detailed breakdown of correct and incorrect answers
- **Quiz Reset**: Option to retake the quiz

## Quiz Questions

1. **Geography**: What is the capital of France? (Paris)
2. **Science**: Which planet is known as the Red Planet? (Mars)
3. **Math**: What is 2 + 2 * 4? (10)

## Setup Instructions

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. **Navigate to the project directory**:
   ```bash
   cd quiz-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Open your browser** and navigate to `http://localhost:3000`

## AI Integration Setup

To enable AI feedback functionality, you need to configure the Gemini API:

### Getting a Gemini API Key

1. Go to [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Create a new API key
4. Copy the API key

### Configuring the API Key

1. **Open** `src/App.js`
2. **Find** the fetch request in the `generateAIFeedback` function (around line 95)
3. **Uncomment** and **replace** the Authorization header:
   ```javascript
   headers: {
     'Content-Type': 'application/json',
     'Authorization': 'Bearer YOUR_ACTUAL_API_KEY_HERE'
   }
   ```

### Alternative: Environment Variable (Recommended)

For better security, use environment variables:

1. **Create** a `.env` file in the root directory:
   ```bash
   touch .env
   ```

2. **Add** your API key to the `.env` file:
   ```
   REACT_APP_GEMINI_API_KEY=your_actual_api_key_here
   ```

3. **Update** the fetch request in `src/App.js`:
   ```javascript
   headers: {
     'Content-Type': 'application/json',
     'Authorization': `Bearer ${process.env.REACT_APP_GEMINI_API_KEY}`
   }
   ```

## How It Works

### Quiz Flow

1. **Question Display**: Users see one question at a time with multiple choice options
2. **Answer Selection**: Clicking an answer automatically progresses to the next question
3. **Score Calculation**: After the final question, the app calculates the user's score
4. **AI Feedback**: The app sends quiz results to Gemini API for personalized feedback
5. **Results Display**: Shows final score, AI feedback, and detailed answer review

### AI Feedback Generation

The app constructs a detailed prompt for the Gemini API including:
- User's final score
- Questions answered correctly and incorrectly
- Specific instructions for the AI to act as a friendly tutor
- Request for encouraging, educational feedback

### Fallback Behavior

If the API call fails or no API key is configured, the app displays a friendly fallback message congratulating the user on completing the quiz.

## Technical Details

### State Management
- Uses React hooks (`useState`) for all state management
- Tracks current question, user answers, score, and AI feedback

### Styling
- Tailwind CSS for responsive, modern design
- Gradient background and clean card-based layout
- Smooth transitions and hover effects

### API Integration
- Uses the Gemini 2.5 Flash Preview model
- Implements proper error handling and loading states
- Includes fallback feedback for API failures

## Customization

### Adding More Questions

To add more questions, modify the `questions` array in `src/App.js`:

```javascript
const questions = [
  // ... existing questions
  {
    id: 4,
    question: "Your new question here?",
    choices: ["Choice A", "Choice B", "Choice C", "Choice D"],
    correctAnswer: "Correct Choice"
  }
];
```

### Modifying AI Prompt

To customize the AI feedback style, edit the `prompt` variable in the `generateAIFeedback` function.

## Troubleshooting

### Common Issues

1. **API Key Not Working**: Ensure the API key is correctly formatted and has proper permissions
2. **CORS Errors**: The Gemini API should handle CORS automatically, but check your network settings
3. **Styling Issues**: Make sure Tailwind CSS is properly installed and configured

### Development Tips

- Use browser developer tools to check for console errors
- Test the API call separately using tools like Postman
- Check the Network tab to see if API requests are being made
